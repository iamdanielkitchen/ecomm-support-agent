# Promo codes and discounts

Fieldstone accepts one promo code per order. Codes apply at checkout and appear as a line-item discount on your order summary. If a code is not working, the most common reasons are expiration, item exclusions, or a minimum purchase requirement that has not been met.

## How promo codes work

Enter your code in the promo field at checkout. The discount applies immediately and shows in your order total before payment. You will see the code and discount amount on your order confirmation email.

Only one code can be used per order. If you try to apply a second code, the first one is removed. This includes stacking a percentage-off code with a dollar-off code or combining two separate promotions. Choose the code that gives you the best savings.

Most codes are case-insensitive, but some promotional campaigns require exact entry. If a code is not applying, double-check the spelling and remove any extra spaces.

## Common reasons a code is invalid

**The code has expired.** Promo codes include an end date in the fine print of the offer. After that date, the code no longer works. Seasonal codes typically expire within a few weeks of the campaign launch.

**Your cart does not meet the minimum purchase requirement.** Some codes require a subtotal of $75 or $100 before the discount applies. Final-sale items and shipping fees do not count toward this threshold.

**The code excludes certain items.** Final-sale products—like our Holiday Candle (FG-SAL-001) and Seasonal Tea Towels (FG-SAL-002)—are excluded from most promotions. A few codes also exclude new arrivals or specific product categories. Check the original offer for a list of exclusions.

**You already used a code on this order.** Only one code applies per transaction. If you entered a code earlier in your session, remove it before trying a new one.

## Promo pricing and returns

Items purchased with a promo code are eligible for return under our standard 30-day policy. Your refund is based on the discounted price you paid, not the original list price. If you return part of a multi-item order and the remaining items fall below the promo's minimum purchase requirement, the discount is forfeited and you are refunded only the amount you actually paid for the returned items.

For example: you buy three items totaling $120 and apply a code for $20 off orders over $100. You pay $100. If you return one item worth $40 at list price, your refund is calculated from the $100 you paid, not the $120 subtotal.

Holiday orders placed during extended promotional windows follow the same logic. See our guide on Holiday returns: extended window for details on seasonal return deadlines.

## FAQ

**Can I apply a promo code to a gift card purchase?**  
No. Gift cards are excluded from promo codes and discounts. See our article Gift cards: purchase and redemption for more on how gift cards work.

**What if my code was valid yesterday but does not work today?**  
The code likely expired overnight. Promo end dates are listed in the original campaign email or on the promotion banner. Once a code expires, it cannot be reactivated.

**Can a human support agent apply a code after I place my order?**  
No. Codes must be entered at checkout. We cannot adjust pricing or apply discounts retroactively. If you forgot to use a code, you may cancel and re-order if your items have not yet shipped.
